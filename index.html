Built for real threat detection scenarios

Developed for CY 201 — Intro to Cybersecurity. Applies intelligent classification to identify phishing, injection attacks, port scans, and vulnerable code — the same threat categories I analyzed manually in Wireshark and Snort IDS labs throughout the semester.

📧
Phishing Detection
Identifies spoofed senders, urgency language, suspicious links, and social engineering patterns.
🔗
Malicious URL Analysis
Flags typosquatting, suspicious TLDs, encoded payloads, and redirect chains.
📋
Log Anomaly Detection
Detects brute force patterns, port scan signatures, and unauthorized access attempts.
💻
Vulnerable Code Review
Spots buffer overflows, SQL injection vectors, and unsafe C library calls (gets, strcpy) — covered in CS 380.
